## 构造方法
```text
-当一个对象被创建时，构造方法用来初始化该对象，构造方法和它类的名字相同，但构造方法没有返回值
-通常会使用构造方法给一个类的实例变量赋初值，或者执行其它必要的步骤来创建一个完整的对象
-不管是否定义构造方法，所有的类都有构造方法，因为Java自动提供了一个默认构造方法，默认构造方法的访问修饰符和类的访问修饰符一致
-一旦定义了自己的构造方法，默认的构造方法就会失效
```

## 可变参数
```text
-JDK1.5 开始，Java支持传递同类型的可变参数给一个方法，方法的可变参数的声明如下所示
    typeName... parameterName
    在方法声明中，在指定参数类型后加一个省略号(...)
    一个方法中只能指定一个可变参数，它必须是方法中的最后一个参数，任何普通的参数必须在它之前声明
```

## finalize()方法
```text
-Java允许定义这样的方法，他在被垃圾收集器析构（回收）之前调用，这个方法叫finalize(), 它用来清除回收对象
-例如：可以使用finalize()来确保一个对象打开的文件被关闭了
-一般格式是： 
    protected void finalize() {
        // 这里写终结代码
    }
 ```

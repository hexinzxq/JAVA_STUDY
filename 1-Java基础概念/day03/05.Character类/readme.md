```text
-Character类用于对单个字符进行操作
-Character类在对象中包装一个基本类型char的值
-Character类提供一系列方法操纵字符，可以使用Character的构造方法创建一个Character类对象
    例如：Character ch = new Character('a');
```

## Character类的方法
```text
-isLetter(): 是否是一个字母
-isDigit():是否是一个数字字符
-isWhitespace():是否是一个空白字符
-isUpperCase():是否都是大写字母
-isLowerCase:是否都是小写字母
-toUpperCase():指定字母的大写形式
-toLowerCase():指定字母的小写形式
-toString():返回字符的字符串形式，字符串长度仅为1
```

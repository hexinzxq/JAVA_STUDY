## Java两大数据类型
```text
· 内置数据类型
· 引用数据类型
    -java提供了8种基本类型：6个数字类型（四个整数型，两个浮点型），一种字符类型，一种布尔型
```
### 内置数据类型
```text
byte
    -8位，有符号，以二进制补码表示的整数；
    -最小值：-128（-2^7）
    -最大值：127(2^7 - 1)
    -默认值： 0
    -byte类型在大型数组中节约空间，主要代替整数，因为byte变量占用的空间只有int类型的四分之一
    -例： byte a = 30; byte b = -20;
short
    -16位，有符号，以二进制补码表示的整数
    -最小值：-32768（-2^15）
    -最大值: 32767(2^15 - 1)
    -默认值：0
    -short数据类型也可以像byte节省空间，是int变量所占空间的二分之一
    -例子：short s = 1000; short d = -390
int
    -32位，有符号，以二进制补码表示的整数
    -最小值： （-2^31）
    -最大值：（2^31 - 1）
    -默认值： 0
    -一般整型的变量默认为int类型
    -例子： int a = 20000; int b = -4900
long
    -64位，有符号，以二进制补码表示的整数
    -最小值： （-2^63）
    -最大值： （2^63 - 1）
    -默认值： 0L
    -此类型主要用在需要整数比较大的系统上
    -例子： long a = 10000L; long b = -30000L;(L不区分大小写，避免混淆一般大写)
boolean
    -表示一位的信息
    -只有两个值： true和false
    -默认值： false
    -这种类型仅作为一种标志来记录true/false的情况
    -例子： boolean isNew = true
float
    -单精度，32位，符合IEEE 754标准的浮点数
    -浮点数不能用来表示精确的值，如货币
    -默认值： 0.0f
    -float在存储大型浮点数组的时候可以节省内存空间
    -例子： float f1 = 123.3f
double
    -双精度，64位，符合IEEE 754标准的浮点数
    -浮点数的默认类型是double类型
    -double类型同样不能表示精确的值，如货币
    -默认值： 0.0d
    -例子： double d1 = 7D; double d2 = 7.; double d3 = 8.D; double d4 = 13.2312
char
    -一个单一的16位Unicode字符
    -最小值：\u0000(十进制等效值为0)
    -最大值：\uffff（即为65535）
    -char数据类型可以存储任何字符
    -例子：char letter = 'A'
```

###  引用类型
```text
-在Java中，引用类型的变量很类似于C/C++的指针，引用类型指向一个对象，指向对象的变量是引用变量，这些变量
在声明时被指定为一个特定的类型，比如Employee、Puppy等。变量一旦声明后，类型就不能改变了
-对象、数组都是引用数据类型
-所有引用类型的默认值都是null
-一个引用变量可以用来引用任何与之兼容的类型
-例子： Site site = new Site("baidu")
```

### Java常量
```text
-常量在程序运行时是不能被修改的
-在Java中使用final关键字来修饰常量
-声明方式和变量类似： final double PI = 3.1415926
-虽然常量名可以用小写，但是为了便于识别，通常使用大写字母表示常量
```
```text
-字面量可以赋给任何内置类型的变量，例如：byte a = 23; char a = 'A'
-byte、int、long和short都可用十进制，16进制以及8进制的方式来表示
-当使用字面量的时候，前缀0表示8进制，前缀0x表示16进制，例如：int a = 0144; int b = 0x64
-和其他语言一样，Java的字符串常量也是包含在两个引号之间的字符序列，例如： "Hello jack", "see\n bird"
  "\"This is a dog \""
-Java语言支持的一些转义字符序列
    *\n   换行
    *\r   回车
    *\f   换页符
    *\b   退格
    *\0   空字符
    *\s   空格
    *\t   制表符
    *\"   双引号
    *\'   单引号
    *\\   反斜杠
    *\ddd   八进制字符
    *\uxxxxx   16进制Unicode字符
```

### 自动类型转换
```text
-整型、实型（常量）、字符型数据可以混合运算。运算中，不同类型的数据先转化为同一类型，然后进行运算，
  转换从低级到高级： byte,short,char->int->long->float->double
-数据类型转换必须满足如下规则
    1.不能对boolean类型进行类型转换
    2.不能把对象类型转换为不相关类的对象
    3.在把容量大的类型转换为容量小的类型时必须使用强制类型转换
    4.转换过程中可能导致溢出或损失精度，例如：int i = 128; byte b = (byte)i;byte最大127，溢出！
    5.浮点数转为整数是通过舍弃小数得到，而不是四舍五入
-自动类型转换必须满足转换前的数据类型位数要低于转换后的数据类型，例如：16位的short可以自动转换为
  32位的int
```

### 强制类型转换
```text
-1.条件是转换的数据类型必须是兼容的
-2.格式：(type)value type是要强制类型转换后的数据类型，例如： 
    int i = 15;byte = (byte)i;强制类型转换i为byte类型
```
